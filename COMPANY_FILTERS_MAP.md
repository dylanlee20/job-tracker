## üöÄ Quick Add Commands

To add a new company, you need:
1. Create scraper file in `scrapers/[company]_scraper.py`
2. Add to `services/scraper_service.py` SCRAPERS dict
3. Add URL to `config.py` COMPANY_WEBSITES dict
4. Test with `python3 test_[company].py`
